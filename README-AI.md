## ğŸ—ï¸ Architecture

### Files

- **`README.md`** - The ONLY file you need to edit! Natural Chinese content
- **`data/schema.json`** - JSON schema defining the structure
- **`data/en.json`** - Auto-generated by GitHub AI (English)
- **`data/zh.json`** - Auto-generated by GitHub AI (Chinese)
- **`scripts/generate-json.py`** - Python script that uses GitHub Models API
- **`.github/workflows/generate-json.yml`** - GitHub Action automation
- **`index.html`** - Website template that loads JSON content

### AI-Powered Workflow

1. **Content Source**: Natural README.md in Chinese (no special formatting required)
2. **Automation**: GitHub Action triggers on README.md changes
3. **AI Processing**: GitHub Models API reads and understands the content
4. **Generation**: AI generates structured JSON following the schema
5. **Translation**: AI translates to English automatically
6. **Validation**: JSON files are validated against schema
7. **Deployment**: Generated JSON files are committed and pushed
8. **Display**: Website loads JSON based on user's language preference
